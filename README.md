# bored-and-hungry

## Description
<p> 
    We created a generator that provides users with an activity to do and a recipe to make.
    Insert a number from 1-5 for people in your party and a preparation time and it will generate one for you!
</p>


## Technologies
<ul>
    <li> HTML
    <li> CSS
    <li> JavaScript
    <li> Skeleton
    <li> JQuery
    <li> API
</ul>
